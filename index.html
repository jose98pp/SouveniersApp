📝 Notas de Diseño

🎨 Paleta de Colores

  • Primario: #d4af37 (Dorado)
  • Secundario: #2c3e50 (Azul oscuro)
  • Fondo: #f8f9fa (Gris claro)
  • Texto: #333 (Gris oscuro)

📱 Componentes Clave

  • Cards: Bordes redondeados, sombras suaves
  • Botones: Esquinas redondeadas, estados hover
  • Formularios: Campos con padding generoso
  • Navegación: Tab bar inferior para móvil
  • Estados: Badges coloridos para pedidos

🔄 Flujos Principales

  • Pedido: Menú → Producto → Carrito → Pago

  • Admin: Dashboard → Gestión → Reportes

⚡ Interacciones

  • Loading: Spinners y skeletons
  • Feedback: Toasts y confirmaciones
  • Animaciones: Transiciones suaves
  • Gestos: Swipe, pull-to-refresh
